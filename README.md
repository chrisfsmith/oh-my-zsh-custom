# oh-my-zsh-custom
Customizations for oh-my-zsh.

Clone the repo and run `<repo>/setup/install.sh` to create `~/.zshrc`.

    $HOME
    └── .oh-my-zsh-custom
        ├── my_lib_patches.zsh
        ├── plugins
        │   └── my_plugin
        │       └── my_plugin.plugin.zsh
        └── setup
        │   └── hostname
        │       └── zshrc
        │       └── zshrc_plugins
        |   └── install.sh
        └── themes
            └── my_awesome_theme.zsh-theme
